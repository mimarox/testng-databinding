The TestNG Data Binding framework provides binding of data from various sources to parameters of TestNG test methods.

Usage Example

As this framework is annotation driven on the user interface side, a small example showing the main components should help getting you started. Consider the following test method and follow the linked annotations. For a general introduction see the {@link net.sf.testng.databinding.GenericDataProvider} class. For detailed information on any single data source, follow the package links given in the data source packages table above. 

	@Test
	{@link net.sf.testng.databinding.DataBinding @DataBinding}
	public void testAdd({@link net.sf.testng.databinding.TestInput @TestInput}({@link net.sf.testng.databinding.TestInput#name name} = "a") final int a, {@link net.sf.testng.databinding.TestInput @TestInput}({@link net.sf.testng.databinding.TestInput#name name} = "b") final int b, {@link net.sf.testng.databinding.TestOutput @TestOutput}({@link net.sf.testng.databinding.TestOutput#name name} = "c") final int c) {
	    assertEquals(a + b, c);
	}
